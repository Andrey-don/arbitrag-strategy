# YT-A — От Госслужбы к Трейдингу: как стать миллионером за год?

## Суть материала

Мотивационное интервью с реальным учеником Клевцова. Владимир — бывший майор полиции, старший следователь — прошёл путь от нуля до нескольких сотен тысяч долларов за год. Нет технических деталей стратегии. Полезно для понимания психологии успешного трейдера и важности подхода.

---

## Путь Владимира: хронология

| Период | Что произошло |
|--------|--------------|
| 2017 | Узнал про Bitcoin от коллеги → купил → случайная прибыль → продал в декабре 2017 |
| 2018 | Потерял ~$10K на ICO (биржа-скам закрылась через 2 дня) |
| 2018–2021 | Изучение теханализа, паттернов, Фибоначи, объёмного анализа — безуспешно |
| 2022 | Курс Ярослава Брина — психологический перелом, смена мышления |
| 2022–2023 | Ещё курсы (включая проп-компанию) — всё ещё безуспешно |
| 2023 | **Точка ноль** — уволился с госслужбы, жена содержала семью |
| 2023–2024 | Курс Клевцова (1-й поток) → арбитраж/алго на MEXC → прорыв |
| Итог | $10 → несколько сотен тысяч долларов за год. Только 2 убыточных дня |

## Ключевые выводы

### Что сработало (в отличие от теханализа)

- **Поиск алгоритмов и неэффективностей** — не рисование графиков
- Торговля на MEXC (арбитраж, «лесенки», алгоритмы роботов)
- Группа из 3 человек → совместный поиск и обсуждение → прорыв
- Дисциплина, педантизм, следование стратегии (навыки из госслужбы перенеслись)
- Параллельно: MEXC (ночью) + российский рынок (днём)

### Что НЕ работает

- Теханализ, паттерны, индикаторы, Фибоначи — потрачено 2–3 года впустую
- «Грааль» мышление — ожидание «кнопки бабло»
- Одиночество без команды/наставника

---

## Психологические факторы успеха

1. **Дисциплина и педантизм** — перенесены с госслужбы
2. **Поддержка партнёра** — Ксения 3 года вывозила финансово
3. **Команда** (группа 3 человек) — разобрали алгоритмы вместе
4. **Постановка целей** — список 100 желаний, цели по деньгам и жизни
5. **«Морковка сзади»** — видел "нижний уровень" жизни, не хотел туда
