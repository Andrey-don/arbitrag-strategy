# р9-13: Клевцов — Обучение новичка, часть 1

## Формат
Видеозапись: Клевцов обучает новичка Филиппа торговле «с нуля» за одну сессию.  
Контент ориентирован на крипто-скальпинг, не MOEX.

---

## Ключевые темы

### Подход к рынку
- Стратегия: «ебать маркетмейкера» — использовать активность маркетмейкера в свою пользу
- Игра с нулевой суммой: либо ты забираешь, либо у тебя забирают + комиссии
- Задача: не «угадывать», а совершать **обдуманные действия**

### Последовательность обучения (по Клевцову)
1. Сначала скальпинг → «спринт, не марафон»
2. Потом арбитраж → «то, на чём я зарабатываю последние 3 года»
3. «Скальпинг нужен для понимания стакана и ленты»

### Инструменты для крипто-скальпинга
- Программы: Tiger Trade, Metascalp, Seascalp, Calibri (скальперские приводы)
- Биржи 3-го эшелона (не Binance/Coinbase): **MEX, GATE, KuCoin, HTX, Bybit, Bitget**
  - Азиатские, лояльные к любым юрисдикциям
  - Регистрация простая, KYC лайтовый
- Инструменты: не биткоин (слишком популярен → нет конкурентного преимущества)
  - Нужны «непопулярные» активы, где конкуренция меньше

### Психология
- Главный барьер для новичка: страх потери, неспособность нажать кнопку
- Практика важнее теории
- Финансовый результат новичка за сессию: **+25% от капитала**

---

## Релевантность для MATS
**Низкая.** Контент — крипто-скальпинг для новичков.  
Для MOEX-арбитража практически неприменим.

**Что полезно:** подтверждение логики обучения Клевцова (скальп → арбитраж), понимание его мышления как автора курса.

---

## Оценка
**Плюсы:** живое обучение, понятная методика для новичков  
**Минусы:** крипто, не MOEX; очень базовый уровень; много «воды»  
**Вывод:** пропустить для MATS, смотреть как введение в стиль автора курса
