# Урок 06: Научил Новичка ТРЕЙДИНГУ за День. Подробное руководство

## 1. Краткое резюме

Трейдер с 15-летним опытом обучает новичка основам скальпинга и сбору спреда на неликвидных токенах с нуля за один день. К концу сессии новичок совершил ~100 сделок и заработал **$0.40** на рабочий объём **$2** (доходность ~25% к активному торговому капиталу). Ни одной убыточной сделки — потому что он не угадывал тренд, а математически собирал разницу между bid и ask внутри зажатого спреда «глупого» биржевого алгоритма (маркетмейкера).

## 2. Полный алгоритм с нуля

### 2.1 Регистрация и настройка биржи

- **Биржа:** MEXC (третий эшелон, лояльны ко всему миру, нет жёстких санкционных ограничений)
- **KYC/верификация:** [НЕЯСНО — уточнить через NotebookLM]
- **Пополнение:** USDT в сети **TRC20** («сеть терка»), с Trust Wallet. Стартовая сумма: **$50** (общий баланс упоминается **$89**)
- **Кошелёк:** средства на спотовом кошельке

### 2.2 Установка и настройка программ

**Терминалы (на выбор):** Tiger Trade, Metascalp, **CScalp (Калибри)**

**Подключение API-ключей:**
1. Биржа → Управление API → создать ключи
2. Права: View account details + View order placing + **Торговля**
3. **Вывод (Transfer) — ЗАПРЕЩЁН**
4. Для фьючерсов: шестерёнка → Preferences → Position mode → **One-way mode**
5. В браузере нажать **F12** → Application → скопировать свой UID
6. В терминале: Добавить подключение (отдельно Spot и Futures) → вставить API + UID
7. Экспортировать настройки через кнопку «Экспорт настроек»

### 2.3 Настройка рабочего места

**Мышка:**
- Левая кнопка → лимитная заявка на **покупку** (buy)
- Правая кнопка → лимитная заявка на **продажу** (sell)
- DPI/чувствительность: [НЕЯСНО — уточнить через скриншот настроек]

**Клавиатура:**
- **Пробел** → мгновенная отмена всех выставленных ордеров
- **Alt** → захват всего остатка позиции для полного закрытия без дробей

**Терминал (открытые панели):**
- Стакан ордеров (зелёная зона — bid, красная — ask)
- Лента тиков (история сделок: кружочки с объёмами и ценами)
- Панель выбора рабочего объёма в долларах
- Графики **отключены** — не нужны

## 3. Детальный алгоритм скальпирования

### 3.1 Выбор инструмента

- Токен: «всратый» (например, **PLB/USDT**) — без реального комьюнити и инвесторов
- **Критерии в стакане:**
  - Широкий спред между лучшим bid и ask
  - В стакане никого нет — только один биржевой бот, который бьёт сам в себя одинаковыми объёмами (например, **$27** или **$2**)
  - График рисует «ёрш» — цена ходит в узком горизонтальном коридоре вверх-вниз

### 3.2 Алгоритм одной сделки (пошагово)

1. Установить рабочий объём в терминале: **$1–2**
2. Навести курсор в **зелёную зону** стакана — чуть ниже текущей цены, перед крупными заявками маркетмейкера
3. **Левая кнопка мыши** → выставить лимитную заявку на покупку
4. Ждать, пока маркетмейкер исполнит заявку
5. Перевести курсор в **красную зону** стакана — ближе к верхней границе спреда
6. **Правая кнопка мыши** → выставить лимитную заявку на продажу
7. Ждать исполнения. Разница в центах остаётся на балансе

### 3.3 Управление открытой позицией

- **Мониторинг:** нижняя панель терминала — текущая позиция и PnL в долларах
- **Передвижение ордера:** маркетмейкер сдвинул диапазон → **Пробел** (отмена) → переставить заявки в новый диапазон
- **Закрытие с остатком:** зажать **Alt** (терминал захватывает весь остаток с дробями) + правая кнопка по красному стакану
- **Досрочный выход:** [НЕЯСНО — уточнить через NotebookLM при каких условиях]

### 3.4 Принципы работы

- **Математика:** купил по 10, продал по 11 → разница 1. Тысяча таких действий = ощутимый доход
- **Роль маркетмейкера:** робот создаёт иллюзию торгов на мёртвом токене, гоняет цену между своими же лимитками. Алгоритм **глупый** — не распознаёт мелких скальперов
- **Малый объём критичен:** заявка на **$100** при боте с шагом **$2** → бот «испугается» и уйдёт. Объём должен быть незаметным

## 4. Разбор результата новичка

- Сделок: **~100**
- Заработано: **$0.40** при объёме **$2** (~25% к капиталу)
- Ни одной убыточной: новичок не угадывал тренд, действовал строго как маркетмейкер — лимитки снизу и сверху, только гарантированный спред

**Ошибки новичка:**
1. Закрывал позицию без Alt → копились микро-остатки токена
2. Жадничал — ставил лимитку слишком близко к центру, цена не доходила. Урок: **«не жадничай, жди»**, ставь ордер дальше

## 5. Что может пойти не так

| Ситуация | Что делать |
|----------|-----------|
| Пришла толпа — алгоритм сломан, стакан раздвинулся | Перестать торговать, найти новый токен |
| Скам токена (делистинг → ноль) | Никогда не держать как инвестицию, только внутри дня |
| Бот сдвинул диапазон котирования | Пробел → переставить заявки |

## 6. Применение в арбитражном боте (Python)

```python
import ccxt.async_support as ccxt
import asyncio

SYMBOL = 'PLB/USDT'
TRADE_SIZE_USD = 2.0  # маленький объём — бот не заметит

async def scalp_market_maker():
    exchange = ccxt.mexc({
        'apiKey': 'YOUR_API_KEY',
        'secret': 'YOUR_SECRET_KEY',
        'enableRateLimit': True,
    })

    try:
        orderbook = await exchange.fetch_order_book(SYMBOL, limit=10)

        # ИСПРАВЛЕНО: bids/asks — список [[цена, объём], ...], берём [0][0]
        best_bid = orderbook['bids'][0][0]
        best_ask = orderbook['asks'][0][0]

        buy_price = best_bid + 0.0001
        amount = TRADE_SIZE_USD / buy_price

        # Левая кнопка мыши → лимитный buy
        print(f"BUY {amount:.4f} {SYMBOL} по {buy_price}")
        await exchange.create_limit_buy_order(SYMBOL, amount, buy_price)

        # TODO: заменить sleep на websocket (ccxt.pro) для отслеживания исполнения
        await asyncio.sleep(2)

        # Alt → захват всего остатка (fetch_balance вместо фиксированного amount)
        balance = await exchange.fetch_balance()
        token = SYMBOL.split('/')[0]
        token_balance = balance[token]['free']

        if token_balance > 0:
            sell_price = best_ask - 0.0001
            # Правая кнопка мыши → лимитный sell на весь остаток
            print(f"SELL {token_balance} {token} по {sell_price}")
            await exchange.create_limit_sell_order(SYMBOL, token_balance, sell_price)

    except Exception as e:
        print(f"Ошибка: {e}")
        # Пробел → отмена всех ордеров
        await exchange.cancel_all_orders(SYMBOL)
    finally:
        await exchange.close()

if __name__ == '__main__':
    asyncio.run(scalp_market_maker())
```

> **Важно:** `asyncio.sleep(2)` — временная заглушка. В продакшне заменить на WebSocket через `ccxt.pro` для мгновенного отслеживания исполнения ордера.
