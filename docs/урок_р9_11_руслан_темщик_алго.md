## Позиция по направленной торговле

> «Направленная торговля может быть прибыльной, и алгоритмизация как раз позволяет превратить классическую направленную торговлю в прибыльную.»

**Подтверждено Русланом:**
- Однонаправленная торговля прибыльна — при наличии статистического преимущества
- Ключ: алгоритмизация убирает эмоциональный фактор
- Направленные сделки ≠ случайные: нужна статистическая закономерность

## Про программирование для трейдеров

Вопрос Антона: «Стоит ли трейдерам учить программирование?»

**Ответ Руслана:** Зависит от предрасположенности:
- Если интересно → однозначно да, даёт долгосрочное преимущество
- Если нет → не обязательно, можно нанять программиста
- Программирование = инструмент, не цель

## Применимость для MATS — СРЕДНЯЯ ⭐⭐

| Что взять | Как применить в MATS |
|---------|--------------|
| Allora / MooseBird как источник данных | Альтернатива MOEX ISS для исторических данных |

**Вывод:** Интервью подтверждает правильность архитектурного выбора MATS: алгоритмизировать рутинный мониторинг → убрать эмоции → применять статистику. Руслан — наглядный пример того, что может вырасти из нашего проекта через 2–3 года.
