# Раздел 9.03 — Синтетические облигации против ипотеки (Михаил)

## Суть материала

Стрим с Михаилом — алготрейдером из группы KasCapital. Тема: стратегия «синтетических облигаций» на MOEX как способ генерации стабильного дохода для гашения ипотечных платежей.

---

## Что такое синтетическая облигация

**Формула:**
```
Синтетическая облигация = Купить акцию (спот) + Шорт фьючерса на ту же акцию
```

**Механика:**
- Ценовой риск акции полностью нейтрализован шортом фьючерса
- Позиция зарабатывает **базис** (разница цены фьючерса и спота) + безрисковую ставку
- Базис схождения к экспирации — предсказуемый доход
- Аналог: как купить облигацию, но через акцию + фьючерс

**Почему "синтетическая":**
- Нет реальной облигации — инструмент создаётся искусственно из двух ног
- Доходность часто выше реальных ОФЗ/корпоративных облигаций

---

## Ключевые параметры стратегии

| Параметр | Значение |
|---------|----------|
| Инструменты | Акция MOEX + фьючерс на ту же акцию |
| Тип дохода | Базис (contango) + безрисковая ставка |
| Горизонт | До экспирации фьючерса |
| Плечо | РЕПО (кредит брокера под залог акций) |
| Аудитория | Опытные трейдеры с капиталом |

---

## Специфика торговли с РЕПО

- **РЕПО** = кредит брокера под залог купленных акций
- Позволяет наращивать позицию сверх собственного капитала
- Важно: **личный счёт ≠ проп-счёт** — условия (комиссии, репо-ставка) разные
- Риск плеча: при большом движении базиса — маржин-колл по одной из ног
- НЕ для новичков: нужен капитал и понимание маржинальных требований

---

## Налогообложение

- Физлицо: **НДФЛ 13–15%** на доход
- ИП: **УСН** (6% доходы или 15% доходы-расходы)
- Актуально при выборе типа счёта

## Михаил как спикер

- Из группы KasCapital (Антон Клевцов)
- Специализация: скальпинг + арбитраж + **алготрейдинг**
- Написал скринеры и инструменты для мониторинга маржи на обеих ногах
- Автоматизация: бот сообщает сумму маржи на каждой ноге (не цену — деньги)
- Хороший пример того, что нужно автоматизировать в MATS

---

## Применимость для MATS

**Прямая применимость:** Средняя

| Что взять | Как применить |
|---------|--------------|
| Идея синт. облигации | Дополнительная стратегия на MOEX (акция + фьючерс) |
| Контроль маржи по деньгам | Алерт «маржа ноги X упала ниже Y рублей» |
| Личный vs проп счёт | Учитывать при настройке комиссий в конфиге |

**Вывод:** Синтетические облигации — это самостоятельная MOEX-стратегия, которую можно добавить в MATS как второй режим (помимо ТАТок). Требует отдельного модуля расчёта базиса и управления РЕПО-плечом.
