## Рабочие инструменты Каширина

**Торгует ТОЛЬКО валюту:**
```
Юань обычный  (CNY/RUB)  + Юань ТОТ
Доллар обычный (USD/RUB)  + Доллар ТОТ
Евро обычный   (EUR/RUB)  + Евро ТОТ
```

Акции и фьючерсы — только при новостных движениях, не повседневно.

**Причина выбора валюты:**
- Высокая ликвидность → понятный стакан
- Для его объёмов — акции уже неинтересны

## Основной паттерн: Айсберги на юане

**Что такое айсберг на юане:**
- Крупная скрытая заявка, видна как «плотность» в стакане
- Две плотности по 10 000 лотов = **«практически железобетонная точка для входа»**

**Алгоритм входа:**
1. Прошло сильное движение (~150 пунктов на юане)
2. Внизу: лонговый айсберг
3. Сверху: ещё один айсберг → ждём его «проедания»
4. Вход: **по рынку**, весь объём, как только видим сильные принты в стакане
5. Стоп-лосс: **2–3 пункта**
6. Тейк: **6–10–15 пунктов** (соотношение 1:3 и выше)

**Стиль:** Агрессивный пробойщик. Все входы/выходы по рынку (не лимитки).

**Важно:** При большом объёме — выход только вручную (рыночный стоп сдвинет цену).

## Параметры управления позицией (юань)

| Параметр | Значение |
|---------|---------|
| Стоп-лосс | 2–3 пункта |
| Комиссия на юане | ~2 пункта (по рынку) |
| Минимальный тейк | 6 пунктов (2× стоп) |
| Оптимальный тейк | 10–15 пунктов |
| Соотношение R:R | 1:3 — 1:5 |
| Стиль входа | По рынку (пробой) |
| Инструмент стопа | Только руками (объём большой) |

## Хеджирование через форекс (USDCNH)

**Инструмент:** USD/CNH фьючерсы (форекс на юань)
- Используется третий год
- Защита от системного риска валютных позиций
- Понимание корреляций между MOEX-юанем и USDCNH — основа стратегии

**Практический смысл:**
- Юань на MOEX и юань на форексе не идентичны → возникает «нейтральная» позиция
- При большом объёме на MOEX → хедж через форекс снижает риск разрыва

## Переход от скальпинга к арбитражу

> «Переход к арбитражу был сознательным решением, направленным на повышение качества жизни, а не только на увеличение прибыли.»

**До арбитража:** Скальпинг — быстро, нервно, требует постоянного присутствия.

**После перехода:**
- Меньше стресса
- Торговля с мобильного телефона (возможно удалённо)
- Не нужно понимать все движения рынка — просто реагировать на них
- Доволен балансом «прибыль / комфорт»

> «Я не стремлюсь понять причины всех рыночных движений. Просто реагирую: цена разошлась — пора входить.»
