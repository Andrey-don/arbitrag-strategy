# Урок 04: Топ 7 Приложений Трейдера. Какие нужно изучать

## 1. Краткое резюме

Трейдер разбирает свой основной рабочий сетап из 7 инструментов, образующих единый фреймворк: от визуализации неэффективностей и контроля стакана до хеджирования и автоматических сигналов. Для серьёзной торговли необходимы инструменты, дающие доступ к микроданным (стакан, лента тиков, кластеры) и мгновенную скорость реакции. Настоящий трейдинг — это «верный процесс забирания с рынка по $1 миллион раз», а не поиск «1000 иксов».

---

## 2. Список приложений

### 1. TradingView
- **Задача:** построение графиков спредов, установка алёртов, агрегация рыночных данных.
- **Почему:** снимает нагрузку по отслеживанию рынка, бесконечное количество вкладок и лейаутов.
- **Стоимость:** от **$150–180** (по скидкам на ЧП/Киберпонедельник/Рамадан) до **$480/год**. Достаточно базовой **Pro**-версии.
- **Автоматизация:** некоторые брокеры позволяют торговать прямо из TradingView. [НЕЯСНО: уточнить про API для ботов]

### 2. Привод Бондаря
- **Задача:** торговля на Мосбирже, доступ к микроданным (стакан, лента, кластеры).
- **Почему:** уникальная эргономика для работы со стаканом, мгновенные лимитные заявки и удары по рынку.
- **Автоматизация:** подключается к проповскому AdminApp для статистики и трансляции рыночных данных.

### 3. MetaTrader 5 / cTrader
- **Задача:** открытие второй «ноги» арбитража на Форекс (хеджирование).
- **Почему:** используется исключительно для контроля баланса (например, **+6 млн руб** на российской стороне, **-50 000 USDT** на Форексе) и исполнения хеджа. Графики в нём не нужны.

### 4. CScalp / Metascalp
- **Задача:** торговля на криптобиржах — стакан и лента в реальном времени.
- **Почему:** уникальный интерфейс для скальпинга и арбитража на крипте, моментальная реакция.
- **Автоматизация:** ✅ подключается через API к **~15 криптобиржам** второго и третьего эшелона (MEXC, Bitget, Bybit, Gate).

### 5. Google Chrome Remote Desktop
- **Задача:** экстренный доступ к рабочему ПК и терминалам с телефона.
- **Почему:** торговать с телефона неудобно из-за мелких пикселей, но необходимо для экстренного управления позицией вдали от рабочего места.
- **Автоматизация:** ❌ нет.

### 6. Telegram
- **Задача:** получение автоматических сигналов от скринеров, обучение, трансляции.
- **Почему:** превратился из мессенджера в торговый инструмент. Серверы-скринеры агрегируют данные и присылают сигналы (тикер, биржа, спред, фандинг) в каналы.
- **Автоматизация:** ✅ используется для создания ботов-скринеров.

### 7. Мобильные приложения банков / бирж / Trust Wallet
- **Задача:** переводы, аутентификация, подтверждение операций, хранение крипты.
- **Почему:** базовая инфраструктура для завода/вывода средств и связи с обменниками.

## 4. Важные замечания

- **Микроданные критичны:** без стакана, ленты тиков и минимальных таймфреймов — нельзя зарабатывать на неэффективностях. Обычные свечи не покажут манипуляции маркетмейкера.
- **«1000 иксов» — ловушка:** трейдинг — это «верный процесс забирания с рынка **по $1 миллион раз**», а не угадывание ракеты.
- **Дневник трейдера не нужен:** в арбитраже статистика «безупречная», достаточно держать в уме общие цифры баланса.
- **Скидки на TradingView:** ~**30 дней в году** (Чёрная пятница, Киберпонедельник, Рамадан) — подписки суммируются, глупо платить $480.

## 6. Применение в арбитражном боте

**Бот полностью заменяет:**
- CScalp / Metascalp — асинхронное чтение стаканов и выставление ордеров через ccxt
- Telegram-скринеры — бот сам сканирует пары и присылает алёрт через Telegram API
- Ручной мониторинг баланса — `fetch_balance()`

**Остаётся ручным:**
- Визуальный анализ исторического спреда в TradingView
- Физические переводы средств между биржами

```python
import ccxt.async_support as ccxt
import asyncio

async def bot_replaces_cscalp_and_screener():
    exchange = ccxt.mexc({'enableRateLimit': True})
    symbol = 'PLB/USDT'

    try:
        ob = await exchange.fetch_order_book(symbol, limit=10)
        best_bid = ob['bids'][0][0] if ob['bids'] else None
        best_ask = ob['asks'][0][0] if ob['asks'] else None

        if best_bid and best_ask:
            spread_pct = (best_ask - best_bid) / best_bid * 100

            if spread_pct > 1.5:
                # Замена Telegram-скринера — уведомление через Telegram Bot API
                print(f"[Alert] Спред на {symbol}: {spread_pct:.2f}%")
                # await telegram_bot.send_message(chat_id, f"Спред: {spread_pct:.2f}%")

    except Exception as e:
        print(f"Ошибка: {e}")
    finally:
        await exchange.close()

if __name__ == '__main__':
    asyncio.run(bot_replaces_cscalp_and_screener())
```
