### Урок 01 — Макроэкономика для криптотрейдеров

**Что это:** Связь биткоина с фондовым рынком США и макроэкономическими циклами. Институционализация крипторынка, режимы Risk-on/Risk-off, ключевые индикаторы.

**Ключевые цифры из урока:**
- Корреляция биткоина с Рассел 2000: **84%** (пик **93%**)
- Корреляция биткоина с Nasdaq: **72%** (пик **86%**)
- Корреляция биткоина с медью: **70%** (пик **94%**)
- Институционалы: **25%** всех сделок, ритейл: **8%**
- Фондовый рынок **80% времени** торгуется на опережающих макрофакторах

**Плюсы:**
- Даёт системное понимание «почему крипта растёт/падает» — без этого слепо торговать
- Конкретные индикаторы для отслеживания: ставка ФРС, М2, кривая доходности, кредитный спред, IPO-календарь
- Честно объясняет: биткоин — это не «цифровое золото», а High Beta Small Cap циклический актив
- Важный практический вывод: в выходные Уолл-стрит волатильность биткоина затухает

**Минусы / ограничения:**
- Урок про среднесрочную торговлю, не про арбитраж — применимость к боту косвенная
- Все «что делать боту» остались без ответа — нет конкретики для автоматизации
- Нет кода, нет API, нет конкретных источников данных для программного получения индикаторов

**Применимость в боте:** Низкая (прямая) / Высокая (контекстная)

**Вывод:** Обязательный фундамент для понимания рыночного контекста. Даёт понять КОГДА арбитражный бот будет работать хорошо (Risk-on, ликвидность высокая, волатильность умеренная) и КОГДА нужно снижать активность (Risk-off, выходные, перед важными макрособытиями). Прямого кода для бота нет — это урок «картины мира».

## Матрица рисков арбитража

_Заполняется по мере изучения уроков. Сейчас — базовые риски любого арбитража._

| Риск | Вероятность | Влияние | Как митигировать |
|------|------------|---------|-----------------|
| Спред исчезает до исполнения | Высокая | Критично | Скорость: websocket + asyncio |
| Комиссии съедают прибыль | Высокая | Высокое | Порог входа > комиссии × 2 |
| Частичное исполнение ордера | Средняя | Высокое | Проверять filled qty |
| Сбой API одной биржи | Средняя | Критично | Retry с backoff, хеджирование |
| Разный объём на биржах | Средняя | Высокое | Проверять ликвидность до входа |
| Funding rate (крипта) | Высокая | Среднее | Учитывать в расчёте спреда |

## Главные выводы

_Раздел заполняется после обработки всех уроков._

### Что НЕ делать (красные флаги)

- Запускать бот на реальные деньги без проверки в DRY_RUN минимум 1-2 недели
- Игнорировать скорость исполнения — в арбитраже миллисекунды решают всё
- Не учитывать комиссии в расчёте минимального спреда для входа
- Оставлять бот без watchdog и автоматического стопа по убытку
