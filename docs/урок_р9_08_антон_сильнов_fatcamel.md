# Раздел 9.08 — Интервью с Антоном Сильновым, главой FatCamel

## Суть материала

Интервью с Антоном Сильновым — главой торгового сообщества FatCamel. Разговор о торговой философии, психологии, подходе к рынку. Меньше конкретных параметров, больше «как думать о трейдинге».

---

## Профиль спикера

- Глава сообщества **FatCamel**
- Торгует: **MOEX (Мамба)** + крипту (скальпинг ~3 года)
- Стратегии: классические сделки + активности + **арбитраж** (MOEX и крипта)
- Кредо: выбрал стратегии для **максимального эмоционального комфорта**

---

## Ключевые мысли о торговле

### Комфорт важнее доходности

> «Максимально комфортно для себя стратегии выбрал и торгую так, чтобы не было просто, потому что комфорт именно эмоциональный для меня важнее, чем финансовый.»

**Вывод:** Если стратегия нервирует — она неправильная для тебя, даже если прибыльная.

### Психология и дисциплина

- Ведёт **торговый дневник** — записывает и анализирует свои ошибки
- Фокус на конкретной области: распыление = убытки
- Личный опыт и ошибки — ценнейший капитал

> «Будьте осторожны. Опыт и ошибки — это ценнейшие уроки.»

### Подход к стратегиям

- Торгует направленные сделки + арбитраж — оба метода сосуществуют
- Технический анализ + работа с объёмами
- Не привязан к одному инструменту — переключается по рынку

---

## Ремарки о рынке

- Торговал Мамбу, Америку, крипту в разное время
- Сейчас фокус: Мамба + крипта (скальпинг)
- «Ретроактивы», «новостные движения» — дополнительные возможности
- Не зацикливается на одном инструменте — следует за возможностями

## Применимость для MATS

**Прямая применимость:** Низкая (больше философия)

| Что взять | Как применить |
|---------|--------------|
| Торговый дневник | Логирование всех сделок MATS с аналитикой |
| Эмоциональный комфорт | Параметры входа должны быть «комфортными» по рискам |
| Дефокус = убытки | Бот торгует только свои стратегии, никаких «ручных» вмешательств |

**Вывод:** Интервью ценно для формирования правильного мировоззрения. Главный тезис: **эмоциональный комфорт в торговле важнее максимального дохода** — это напрямую влияет на параметры риск-менеджмента MATS. Автоматизация убирает эмоции → одна из ключевых причин делать бота.
