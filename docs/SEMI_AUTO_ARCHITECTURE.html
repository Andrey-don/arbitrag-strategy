Схема потока данных

Компоненты системы

📡

Monitor

  • Опрашивает T-Invest API каждые 5 сек
  • GetLastPrices — TATN + TATNP
  • GetCandles — история для контекста
  • Передаёт данные в Signal Engine
🧠

Signal Engine

  • Спред = TATN − TATNP (руб.)
  • Ratio = TATN / TATNP
  • Z-score — статистическое отклонение от нормы
  • Тренд спреда — разворот / рост / спад
  • Выход: LONG_SCALP · LONG_GOOD · EXIT · нет
🚨

Alert Panel

  • Мигающий блок поверх дашборда
  • Параметры: цены, лоты, стоп, прибыль, R/R
  • Кнопки: ✅ РАЗРЕШИТЬ / ❌ ПРОПУСТИТЬ
  • Таймаут 30 сек → автоотмена
  • Опционально: звуковой сигнал
⚡

Order Executor

  • asyncio.gather(buy_leg, sell_leg)
  • Обе ноги одновременно — железное правило
  • T-Invest API: PostOrder
  • Тип: лимитный или рыночный
  • DRY_RUN=true — только лог, без отправки
📊

Position Tracker

  • P&L в реальном времени (руб. и %)
  • Текущий спред vs спред входа
  • Стоп-лосс: авто при −2% депо
  • Тейк-профит: алерт → ЗАКРЫТЬ
  • Счётчик времени в позиции
📁

Logs

  • Файл logs/trades.csv
  • Каждая сделка: время входа/выхода
  • Цены обеих ног, лотность
  • P&L, причина закрытия
  • Основа для анализа стратегии

Как выглядит Alert Panel

🚨 СИГНАЛ: LONG SCALP
Купить TATNP 533.80 ₽ × 3 лота
Продать TATN 563.80 ₽ × 3 лота
Спред сейчас 30.0 ₽
Цель выхода (схождение) 0.5 ₽
Прогноз прибыли +87.9 ₽
Стоп-лосс (−2% депо) −200 ₽
Соотношение R/R 1 : 3.0
Автоотмена через 30 сек

Что означает каждое поле:

Купить TATNP — покупаем дешёвую ногу (прив.)
Продать TATN — шортим дорогую ногу (обычку)

Спред — текущая разница цен. Ждём схождения к 0–0.5 ₽

Прогноз — (спред − 0.5) × лоты − комиссии

Стоп — фиксированный риск по депо (не по цене инструмента)

R/R — соотношение риска к прибыли. Ниже 1:2 — не входить.

Таймаут 30 сек — если не нажал, сигнал снимается. Нет времени — нет сделки.

Алгоритм работы — шаг за шагом

1

Monitor: получить цены

Каждые 5 сек → T-Invest GetLastPrices для TATN и TATNP. Параллельно GetCandles за N минут для исторического контекста.

2

Signal Engine: вычислить показатели

Спред, Ratio, Z-score за последние 50 свечей, тренд спреда. Если спред ≥ 5 руб. + ratio ≥ 1.008 + z-score > 1.5 → генерировать сигнал.

3

Alert Panel: показать параметры

Рассчитать лотность (риск-менеджмент), прогноз прибыли, R/R. Показать мигающий блок. Запустить таймер 30 сек.

4а

Трейдер нажимает РАЗРЕШИТЬ → Order Executor

asyncio.gather(buy_TATNP, sell_TATN) — обе ноги одновременно. Лимитные ордера на bid/ask. Логировать в trades.csv.

4б

Трейдер нажимает ПРОПУСТИТЬ или истёк таймаут

Сигнал отменяется. Система продолжает мониторинг. Запись в лог: «сигнал отклонён».

5

Position Tracker: вести позицию

Показывать P&L каждые 5 сек. Отслеживать спред: цель схождения 0.5 ₽. При достижении тейк-профита → алерт ЗАКРЫТЬ.

6

Выход из позиции

Тейк: алерт трейдеру → [ЗАКРЫТЬ] → asyncio.gather(close_buy, close_sell).
Стоп: автоматически при −2% депо (без подтверждения). Безопасность важнее.

Расчёт параметров входа (риск-менеджмент по Виктору)

Параметр Формула Пример (депо 10 000 ₽)
Макс. просадка 20% от депо 2 000 ₽
Частей входа 5 частей (усреднение) —
Риск на 1 сделку просадка / 5 400 ₽
Стоп-лосс (спред) вход + 3 руб. 33.0 ₽ (если вошли при 30)
Лотность риск / (стоп × 1 лот) 400 / (3 × 1) ≈ 133 лота
Тейк-профит стоп × 3 (R/R = 1:3) 9 руб. схождения → 0.5 ₽
Прогноз прибыли (спред − 0.5) × лоты − комиссии ≈ +1 200 ₽ (без гарантий)
