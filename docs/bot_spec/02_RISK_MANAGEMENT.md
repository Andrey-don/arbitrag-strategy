## Лимиты позиции

### ТАТки (MOEX)

| Лимит | Правило |
|-------|--------|
| Начальный объём | 1–5 лотов (учебный) |
| Рабочий | до 1 000 лотов (безопасный) |
| Предел маржи | Не превышать 60% депозита в позиции |
| Максимум добавок | По мере снижения спреда, малыми частями |

### Предупреждение о маржине

```
Если (используемая маржа / депозит) > MARGIN_WARNING_THRESHOLD:
    → Уведомить оператора
    → Прекратить добавление позиции

Если (используемая маржа / депозит) > MARGIN_CRITICAL_THRESHOLD:
    → НЕМЕДЛЕННО начать закрытие позиции ВРУЧНУЮ
    → Не ждать автоматического margin call брокера
```

**Реальный кейс (урок_15):** маржин-колл = −40% депозита.
Когда брокер закрывает принудительно — очень сильный сквиз.
**Закрывать вручную ДО того, как брокер начнёт принудительное закрытие.**

## Правила управления позицией

### Что разрешено

| Действие | Условие |
|---------|--------|
| Усреднение (добавка) | ОБЯЗАТЕЛЬНО обе ноги одновременно |
| Частичное закрытие добавок | На волатильности (0.3+ руб движение вверх) |
| Удержание основной позиции | До целевого уровня (5–20 руб) |
| Перенос через ночь | Только если есть финансовый смысл vs стоимость переноса |

### Что запрещено

| Запрет | Почему |
|--------|--------|
| Закрыть одну ногу без второй | Создаёт направленную позицию — это уже не арбитраж |
| Усреднять только убыточную ногу | Нарушает нейтральность, ломает логику коинтеграции |
| Ждать margin call брокера | Принудительное закрытие = гарантированный сквиз |
| Держать unbalanced position | Любая несбалансированность = риск |

## Торговля «у единицы» (ratio ≈ 1)

Спред около нуля = зона повышенной волатильности, начало потенциального большого движения.

```
if spread < 0.5:  # "у единицы"
    volatility_multiplier = 1.5  # расширить ожидаемый диапазон
    prefer_long_spread = True    # лонг безопаснее шорта
    extra_margin_reserve = True  # держать дополнительный запас
```

**Наблюдение (урок_15):** «шорт брать опаснее, чем лонг. Тем более мы у единицы — это начало своего. Повышенная волатильность.»

## Правила по направлению (шорт vs лонг спреда)

```
Лонг спреда (ЛОНГ TATN + ШОРТ TATNP):
  - Риск снизу ограничен (спред исторически не уходит < 0)
  - Исторический пол: ~0 руб (кроме ковид: −6 руб)
  - РЕКОМЕНДУЕТСЯ для основной позиции

Шорт спреда (ШОРТ TATN + ЛОНГ TATNP):
  - Риск неограничен вверх (спред может уйти на 20+ руб)
  - Использовать только при явном расширении спреда
  - ЗАПРЕЩЁН при спреде < 1 руб (у единицы)
```
