### Claude Code + TradingView — как это работает

Трейдеры подключают Claude к TradingView через **Webhooks**:

```
TradingView алерт (Pine Script условие сработало)
  → POST-запрос на Webhook URL
  → Python сервер получает сигнал
  → передаёт контекст в Claude API
  → Claude анализирует → возвращает решение
  → Python выставляет ордер
```

**Ограничения:** TradingView Webhooks — только с подпиской PRO и выше.

**Наша схема проще и не требует TradingView PRO:**

```
T-Invest API → данные каждые 500мс
  → calc_spread_signals() → сигнал
  → (опционально) Claude API → анализ сложных ситуаций
  → T-Invest API → выставить лимитный ордер
```

Claude в нашей архитектуре — **не в критическом пути** торговли (LLM слишком медленный для каждой сделки). Применим для: анализа паттерна участника, оценки новостного контекста, аномальных ситуаций.

## Брокер и исполнение

### T-Invest / Тинькофф (основной — РЕШЕНО)
- **Тип:** брокерский счёт с полным API
- **Статус:** ✅ счёт открыт, баланс 10 000 ₽, API токен создан
- **SDK:** `tinkoff-invest-python` (официальный, pip install)
- **Что умеет:** получать котировки, выставлять лимитные/маркет ордера, получать позиции, историю сделок
- **Акции ММВБ:** ✅ TATN, TATNP поддерживаются
- **Ограничение:** 10 000 ₽ — только DRY_RUN; для реальной торговли ТАТками минимум 100 000 ₽

```python
# Подключение (токен из .env)
from tinkoff.invest import Client
with Client(token=TINKOFF_TOKEN) as client:
    # получить цены
    # выставить ордер
    pass
```

### CScalp (для ручного исполнения)
- **Терминал:** Привод (специализированный стаканник)
- **Статус:** ✅ установлен, подключён через T-Invest API
- **Назначение:** ручная торговля (стакан, быстрые лимитки)
- **API для автоматизации:** ❌ нет публичного API
- **Горячие клавиши:** Escape = отменить все лимитные заявки
- **Вывод:** для автомата не нужен — ордера выставляются напрямую через T-Invest API

### Диверсификация брокеров (из урок_16)
- Несколько пропов = правильно (если маржит в одном — перекидываешь)
- Разные пропы = разные таблицы переносов, разные лимиты
- ЗАПРЕТ: не торговать одновременно через разные счета одного брокера по одному инструменту
