# Урок 05: Худшие стратегии трейдинга, на которых теряют деньги

## 1. Краткое резюме

В этом видео трейдер с 15-летним опытом разбирает популярные торговые стратегии из интернета и объясняет, почему большинство из них являются провальными. Основной посыл: настоящая стратегия должна эксплуатировать рыночную **неэффективность** (например, арбитраж), а не пытаться угадать направление движения цены. Для арбитражного бота это отсекает желание программировать логику на основе запаздывающих индикаторов, технического анализа или трендов — фокус только на математически обоснованном сборе спреда.

---

## 2. Разбор провальных стратегий

- **Интуиция** — торговля на ощущениях. Нет чётких шагов, рисков, точек входа/выхода. Отрицательная доходность.

- **Трендовая стратегия** — попытка определить тренд и следовать за ним. Тренд можно идентифицировать только после формирования — это казино.

- **Range Trading** — торговля от границ диапазона. Постфактумный анализ. Исключение: фундаментально ограниченные валюты (рубль **50–120**).

- **Мартингейл как самостоятельная стратегия** — удвоение позиции после убытка без реального торгового преимущества. Экспоненциальный рост убытка до потери депозита. Работает только как вспомогательный инструмент управления позицией внутри рабочей стратегии.

- **Квадрохедж / Замок** — одновременная покупка и продажа одного инструмента на одном счёте. Иллюзия безопасности. Всё равно требует угадывать направление + огромные комиссии брокеру. *«Услышите замок — убегайте».*

- **Smart Money** — поиск следов «кита» на графике. Крупные игроки не оставляют очевидных следов. На каждую покупку есть продажа — объём не указывает направление.

- **Мемкоины** — покупка хайповых токенов. Мусорные активы, которые неизбежно будут стоить **ноль**.

- **Алготрейдинг на индикаторах** — боты на пересечении скользящих средних и т.д. Из созданных миллиардов таких роботов ни один стабильно не работает — индикаторы не описывают рыночные неэффективности.

## 3. Механика провала (почему теряют деньги)

| Стратегия | Причина провала | Типичный убыток |
|-----------|----------------|----------------|
| Трендовая | Невозможность предсказать будущее, торговля постфактум | [НЕЯСНО] |
| Мартингейл | Экспоненциальный рост убытка до потери депозита | 100% депозита |
| Квадрохедж (Замок) | Комиссии + необходимость угадывать точку расфиксации | [НЕЯСНО] |
| Мемкоины | Актив не имеет ценности, обесценивается | До **0** |
| Алготрейдинг на индикаторах | Индикаторы не описывают рыночные неэффективности | [НЕЯСНО] |
| Smart Money | Ложная интерпретация: на покупку всегда есть продажа | [НЕЯСНО] |

## 4. Как трейдер отличает рабочую стратегию от провальной

**Признаки рабочей стратегии:**
- Чётко описана конкретная неэффективность или триггер (расхождение спреда, событие, период)
- Есть ответы на вопросы: какие шаги → какой риск → точка входа → условия выхода

**Красные флаги (сигналы провала):**
- Слово **«замок»** — немедленно уходить
- Попытка угадать направление («сейчас вырастет»)
- Подмена понятий: стиль торговли (скальпинг, свинг) выдаётся за стратегию

## 5. Типичные ошибки новичков

- **Путаница в терминах:** называть «скальпинг» или «свинг-трейдинг» стратегиями. Это стили удержания позиции — внутри них должна быть реальная стратегия.
- **Попытка «торговать неэффективностью»** буквально (купить её). Нужно извлекать прибыль ИЗ неэффективности, как в арбитраже.
- **Слепая вера в технический анализ:** линии поддержки/сопротивления, паттерны («голова и плечи») выглядят идеально на истории, но в моменте пробития трейдеров «рвёт». Работает не паттерн — работает сам трейдер.

## 6. Применение в арбитражном боте — защитные механизмы

**Ошибки, которые может совершить бот:**
- Использование индикаторов как фильтра для входа в арбитражную сделку
- Мартингейл для «отыгрыша» убыточной ноги
- Создание «замка» на одной бирже для хеджирования убытка

**Защита:**
- Логика строится исключительно на математической разнице цен (спред / ставка фандинга) — никаких трендов
- Жёсткие стоп-лимиты по времени удержания без мартингейла
- Разнонаправленные позиции только на РАЗНЫХ биржах — никаких замков

```python
async def evaluate_arbitrage_opportunity(exchange_a, exchange_b, symbol):
    # ТОЛЬКО математическая неэффективность — никаких индикаторов
    ticker_a, ticker_b = await asyncio.gather(
        exchange_a.fetch_ticker(symbol),
        exchange_b.fetch_ticker(symbol),
    )
    spread = (ticker_b['bid'] - ticker_a['ask']) / ticker_a['ask'] * 100
    return spread > ENTRY_THRESHOLD  # 3.0% из урока 03

async def execute_trade(spread_valid):
    if not spread_valid:
        return
    # Две ноги на РАЗНЫХ биржах — не замок, а настоящий арбитраж
    await asyncio.gather(
        exchange_a.create_market_buy_order(symbol, TRADE_SIZE),   # лонг
        exchange_b.create_market_sell_order(symbol, TRADE_SIZE),  # шорт
    )
```
