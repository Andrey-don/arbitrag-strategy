# YT-B — Принципы и торговые стратегии успешных трейдеров

## Суть материала

Стрим Клевцова об отличиях успешных трейдеров. Концептуальный уровень. Показаны конкретные примеры неэффективностей (синтетические облигации, арбитраж перп/спот, металлы). Второй после р9-03 источник по теме синтетических облигаций — следующей стратегии после ТАТок.

---

## Определение успешного трейдера

> **Успешный трейдер — тот, кто систематически, понятным образом воспроизводит специфические действия на темах/неэффективностях, и делает это регулярно.**

НЕ тот, кто:
- Случайно поймал буллран или мем-коин
- Торгует по паттернам/теханализу/статистике
- Работает на самых популярных инструментах

---

## Где живут успешные трейдеры

| Принцип | Объяснение |
|---------|-----------|
| **Непопулярные инструменты** | На популярных (BTC, ETH, топ-акции) нет тем и неэффективностей |
| **MOEX сейчас** | После 2022 — меньше иностранцев и крупных участников → больше неэффективностей |
| **Розничный рынок** | Ритейл + проп-компании = много дырок, которые можно использовать |
| **Третий эшелон** | Ещё меньше народу → ещё больше возможностей |

> «Чем популярнее тема — тем меньше шансов там заработать. Толпа всегда не права.»

## Теханализ — не торговля

```
Теханализ = инструмент анализа и сравнения
НЕ = торговая стратегия

Торговая стратегия = конкретные последовательные действия 
  с параметрами + понимание "У кого я отнимаю деньги?"
```

Паттерны работают только в бычьем рынке — создают иллюзию рабочей системы. На медвежьем рвут в хлам.

**Главный вопрос успешного трейдера:** «У кого я отнимаю деньги в этой сделке?»

## Примеры неэффективностей (конкретные)

### 1. Синтетическая облигация (MOEX) ⭐

```
Инструменты: акция (спот) + шорт фьючерса на эту акцию (квартальный)
Пример: акции Газпрома + шорт GAZR

Механика:
  При открытии квартала: разница цена_фьюча > цена_спота на ~3–4%
  К дате экспирации: цены сходятся → разница = ваша прибыль

Доходность: 3–4% за квартал (~12–16% годовых)
Без плеча: ниже ставки ЦБ → невыгодно
С плечами пропа: 40% на свой капитал за квартал (устарело — пропы закрыли дырку)
```

> Примечание: сейчас "дырка с пропами" закрыта. Чистая синтетическая облигация — ~3-4% квартал.

### 2. Металлы (Золото/Серебро) ⭐

```
То же самое: спот металла + шорт фьюча на металл
Особенность: расхождение иногда 7–11% в моменте (пример декабрь)
Скальпировать: надо сидеть в стакане, знать инструмент
Держать до экспирации: проще, меньше требований
```

### 3. Перп/Спот арбитраж (крипта)

```
График: спот / перп ≈ 1.00 (колеблется ±1-3%)
В моменте декабрь: до 25% расхождения
Платит: маркетмейкер (мартос)
Сложность: нужно балансировать ноги, следить за фандингом
```

### 4. Межбиржевой арбитраж (крипта)

```
Один инструмент на разных биржах
График отношения ≈ 1.00
При расхождении → арбитраж
Сложность: перевод средств между биржами
```

## Темы vs Неэффективности

```
Тема — понятие шире, чем неэффективность
Неэффективность — это всегда тема
Тема — не всегда неэффективность (может быть просто паттерн в периоде)

Неэффективность = арбитражная возможность
Её ключевое свойство: НЕИЗБЕЖНОСТЬ события (цены сойдутся)
```

> **«Арбитраж — история с неизбежным понятным концом. У вас гарантированный исход сделки.»**

Это принципиальное отличие от направленной торговли, где вы «играете в угадайку».

---

## Как находить темы

- **Гипер-насмотренность** — невозможно придумать с нуля, надо видеть много раз
- **Коммьюнити 100 глаз** — кто-то найдёт то, что другой не видит
- **Случайность + шило в заднице** — не расслабляться, везде смотреть, искать
- **Нельзя брейнштормить** — темы не рождаются в комнате, их надо находить в рынке

---

## О макроэкономике

- Теханализа недостаточно → макро помогает определить период
- Для арбитражных тем макро менее критично (исход неизбежен независимо от направления)
- Но для синтетических облигаций — период спокойного рынка важен

## Применимость для MATS — ВЫСОКАЯ ⭐⭐

| Что взять | Как применить |
|---------|--------------|
| «Непопулярный» TATNP — меньше ликвидности → больше неэффективностей | Подтверждает выбор пары |
| Синтетические облигации = следующая стратегия | После отработки ТАТок → спот + шорт фьюча на TATN или Газпром |
| «У кого отнимаю деньги?» | В ТАТках: маркетмейкер TATNP (прив «рвётся» сильнее) |
| Арбитраж = неизбежность события | Спред коинтегрированной пары всегда возвращается → обоснование стратегии |
| Теханализ бесполезен | MATS не использует индикаторы — только спред и ratio |
