# MATS Dashboard v1.0

**Дата:** 2026-05-08

## Что реализовано

- Мониторинг цен TATN / TATNP в реальном времени
- Данные: T-Invest REST API (токен из .env)
- Три синхронных графика: цены, относит. спред%, ratio
- Двойная ось на верхнем графике (каждая линия на своей шкале)
- Текущий сигнал: NO_SIGNAL / ALERT / LONG_SCALP / LONG_GOOD / TAKE_PROFIT / EXIT
- Настройки: история (1–10 дн.), таймфрейм (1–60 мин), интервал обновления
- Запуск: start_dashboard.bat (независимо от VS Code)

## Что НЕ реализовано (следующие версии)

- Подтверждение входа кнопкой (полуавтомат)
- Выставление ордеров через T-Invest API
- Трекинг позиции и P&L
- Алерты (звук, уведомление)
- Риск-менеджмент при входе
