# MATS Dashboard v2.0 — Alert Panel (полуавтомат)

**Дата:** 2026-05-08

## Что добавлено

- **Alert Panel** — мигающий блок при сигналах LONG_SCALP / LONG_GOOD
- **Кнопка РАЗРЕШИТЬ** — подтверждение входа (ордера пока не выставляются, v3.0)
- **Кнопка ПРОПУСТИТЬ** — игнорировать сигнал
- **Таймаут 30 сек** — автоотмена если не нажал
- **Cooldown 2 мин** — не спамить алертами после закрытия
- **Расчёт входа** — лотность, прогноз прибыли, риск, R/R по РМ Виктора (р9-09)
- **Константы РМ** — DEPOSIT=10000₽, STOP=+3₽, TARGET=0.5₽, MIN_RR=2.0

## Что НЕ реализовано (следующие версии)

- Реальное выставление ордеров (v3.0)
- Stop Orders на сервер T-Invest при входе (v3.0)
- Position Tracker P&L (v3.1)
