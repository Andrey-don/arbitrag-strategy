# MATS Dashboard — Установка на новый компьютер

## Что это
Дашборд мониторинга спреда ТАТок (TATN / TATNP) в реальном времени.
Данные: T-Invest API. Запускается в браузере.

---

## Шаг 1 — Установить Python

Скачать Python 3.12 или новее: https://www.python.org/downloads/
При установке поставить галочку **"Add Python to PATH"**

---

## Шаг 2 — Распаковать архив

Распаковать `MATS_Dashboard.zip` в любую папку, например `C:\MATS\`

---

## Шаг 3 — Установить зависимости

Открыть командную строку (Win+R → cmd → Enter), перейти в папку:
```
cd C:\MATS
```
Создать виртуальное окружение и установить пакеты:
```
python -m venv venv
venv\Scripts\activate
pip install -r requirements_dashboard.txt
```

---

## Шаг 4 — Создать файл с токеном

В папке `C:\MATS\` создать файл `.env` (через Блокнот или VS Code):
```
TINKOFF_TOKEN=вставить_токен_от_T-Invest
```
Токен получить в приложении Т-Инвестиции → Настройки → Токен API (только чтение).

---

## Шаг 5 — Запустить

Двойной клик на `start_dashboard.bat` → откроется браузер с дашбордом.

---

## Остановить

Закрыть чёрное окно консоли или нажать Ctrl+C в нём.
