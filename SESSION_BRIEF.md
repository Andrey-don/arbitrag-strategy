- MOEX-фильтр добавлен в движок бэктеста (был только в Тренажёре)
- **Hedge ratio**: автоматически вычисляется из медианных цен при загрузке
  - TATN/TATNP, SBER/SBERP → hedge=1 (цены близкие)
  - TGLD/AKGD → hedge≈21 (TGLD≈13₽, AKGD≈270₽)
- Нормализованный спред: `close_a × hedge - close_b` (для золота: TGLD×21 − AKGD ≈ 0₽)
- UI показывает лоты по каждой ноге: `⚖️ TGLD = 21 лот × 13₽ | AKGD = 1 лот × 270₽`
- Заголовок страницы меняется динамически под выбранную пару
- Колонки переименованы: `close_tatn/close_tatnp` → `close_a/close_b` (универсально)

3. Claude даёт пороги на основе этого диапазона

**Важно про TGLD/AKGD:**
- Hedge ratio ≈ 21 (TGLD дешевле: 1 лот AKGD = 21 лот TGLD)
- Спред до hedge: mean=-255₽ (бессмысленный) → после hedge: ≈ 0₽ (нормализованный)
