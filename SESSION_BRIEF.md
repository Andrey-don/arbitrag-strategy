## Ключевые знания для разработки бота

- **Вход:** Telegram-алерт → визуальная оценка истории спреда → одновременный вход
- **Ratio:** `MEXC:BTC / BYBIT:BTC` в TradingView, горизонталь на 1, отклонение ±3% = сигнал
- **Исполнение:** `asyncio.gather(buy, sell)` — две биржи одновременно, < 1 сек
- **Мониторинг:** суммарный PnL по двум ногам → если `+1%` → закрыть обе ноги
- **Параметры:** порог 3–3.5%, плечо ×3–4, размер ноги $100, фандинг 0.3%/отсечку
- **Запрещено в боте:** индикаторы, мартингейл при убытке, замок на одной бирже
