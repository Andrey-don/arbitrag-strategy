## Ключевые знания из уроков (для разработки бота)

- **Вход:** Telegram-алерт от скринера → визуальная оценка истории спреда → одновременный вход
- **Ratio:** `MEXC:BTC / BYBIT:BTC` в TradingView, горизонталь на уровне 1, отклонение ±3% = сигнал
- **Терминал:** Привод Бондаря = два стакана рядом, диагональный клик < 1 сек → `asyncio.gather()`
- **Мониторинг:** суммарный PnL по двум ногам → если `+1%` → закрыть обе ноги
- **Усреднение:** только вручную, автоматику не реализовывать на первом этапе
- **Параметры:** порог входа 3–3.5%, плечо ×3–4, размер ноги $100, фандинг 0.3%/отсечку
