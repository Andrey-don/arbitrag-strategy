## Ключевые знания MOEX-раздела (уроки 11–17)

- **ТАТки (TATN/TATNP):** коинтегрированная пара, спред 2–5 руб (скальп) / до 20 руб (среднесрочно); пол~0 руб; обычка толще стакан, прив «рвётся» сильнее
- **Налоги:** лонг+шорт полностью сальдируются — только чистый результат
- **Брокер:** CScalp (Привод), пропсчёт, реферальная ссылка kas.wtf/referal/KlevcovDIV
- **Автоматизация:** MOEX ISS API бесплатно; мониторинг спреда → алерт Telegram → ручное исполнение (фаза 1)
- **Риски:** маржин-колл (−40% депо был у автора), закрывать ВРУЧНУЮ; шорт спреда опаснее
- **ЗАПРЕЩЕНО:** «бить в свою заявку» (ЦБ следит)
- **Скринер:** готовых нет — писать самому (Python + MOEX ISS)

## Ключевые знания для разработки бота

- **Вход:** Telegram-алерт → оценка истории спреда → asyncio.gather(buy, sell)
- **Ratio:** отклонение ±3% от 1 = сигнал. Порог входа 3–3.5%, плечо ×3–4, нога $100
- **Скальпинг:** объём $1–2, bids[0][0] / asks[0][0], fetch_balance для остатка, cancel_all_orders
- **Запрещено:** индикаторы, мартингейл при убытке, замок на одной бирже
- **Скальпинг (урок 06):** $2/сделка, спред ~0.07 цента, Alt=закрыть позицию, Пробел=снять заявки, маркетмейкер уходит если объём > $100
- **Арбитраж (урок 07):** ratio отклонение ≥3.5%, лонг дешёвый + шорт дорогой, закрыть до фандинга, плечо ×3–4, скринер=Telegram-бот, 10 бирж, ~1%/день
- **Капитал (урок 08):** минимум $10K, распределение 30% спот + 30% фьюч + 40% резерв маржи; до $2K — только лесенки; порог входа 3–5% спред; проверять фандинг перед входом
- **Фандинг-арбитраж (урок 09):** спот BTC + шорт CoinM 1×; базовая ставка 0.01%/8ч = ~11%/год; 70–80% времени положительный; плечо строго 1×; мин. лот $100; диверсификация по биржам Binance/Bybit/OKX; Statement 2021: ROE 15.04%, Winrate 93.2%, Sharpe 2.98
- **Синтетические пары (урок 10):** TradingView → скринер Binance+Bybit → экспорт CSV → Excel (транспонировать, СЧЁТЕСЛИ, удалить дубл.) → СЦЕПИТЬ("BINANCE:";A1;"/BYBIT:";A1;",") → импорт ~400 пар в вотчлист
