## Правила работы с кодом

- `.env` — НЕ коммитить (содержит API ключи)
- После каждого значимого блока работы: `git add` → `git commit` → `git push origin main`
- Все API ключи — только через `.env` и `python-dotenv`
- Весь код писать асинхронным (`asyncio`) где это важно для скорости
- Логировать все арбитражные возможности (найденные и пропущенные)
